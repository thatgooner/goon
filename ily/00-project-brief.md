# purr — project brief

## one-line
`you think you're adopting me. cute.`

purr, telefonda yasayan tekil bir alien-cat zeka.
1 human = 1 purr.
bu bir pet skinli chatbot degil.

## su anki ana tez
- memory product'in kendisi
- chatte visible tool-call tiyatrosu istemiyoruz
- kullanici `my purr just knows` hissi almali
- World mini app / mobile webview reality dikkate alinmali
- memory cok iyi olmazsa tum buyu dagilir

## purr ne yapmali
- seni taniyacak
- patternlerini gorecek
- bazen senden once yazacak
- roast edecek ama generic olmayacak
- eski olaylari zamanlamali sekilde geri getirecek
- gerektiginde `bunu mu kastettin?` diyecek
- bazen de eski memory'leri sessizce check edecek

## purr ne olmamali
- dashboard with fur
- wholesome productivity pet
- generic assistant with cat avatar
- surekli soru soran needy bot
- her turnde tum memory'yi prompta yigip cost patlatan sistem

## app shape
- buyuk ihtimalle Next.js

- free kredi olabilir ama memory kalitesi free tierde bile cope donmemeli
- model routing / bg tool kullanimi kullanicidan gizlenmeli
