## task summary

pressure-tested whether a slower "referee layer" above the deterministic backbone + narrow evaluator would add value.

### verdict: PARTIAL YES — reframed

the concept is useful. the framing is wrong.

"referee layer" implies a new architectural layer. rejected that framing — invites scope creep, unclear authority, and "let it think about memory in general" drift.

the right answer: **worker H in the existing deferred maintenance lane** (note 18 section 3). same contract, same mutation path, same lane. just watching for temporal failure patterns that workers A-G miss because they handle individual items.

### what the gap actually is

existing workers handle individual items/decisions. nobody watches for temporal patterns across events:
- contradiction churn (same topic challenged/superseded >3× in 2 weeks)
- pattern oscillation (promote/demote cycling)
- repeated user corrections on same topic
- evaluator low-confidence clusters
- stale truth surviving in packs across multiple rebuilds
- proactive veto repeats on same topic cluster

all detectable with SQL queries. no LLM needed for detection.

### runtime form

scheduled typed audit process. not a subagent. not a one-shot reviewer.
- SQL-first detection
- typed `audit_finding` structs as output
- recommendations feed into existing workers (B, C, D, E, proactive gate)
- never writes truth directly
- never runs per-turn
- never accesses raw conversation text

### subagent verdict: NO

true subagent is overkill. high cost, scope creep risk, needs its own context window, debug is harder. the gap can be filled with SQL queries + typed findings.

### Hermes dogfood impact

after phase 3, not before. early phases don't have enough events for temporal pattern detection. would add noise.

### build order impact

new parked slice 5 (`memory-health-auditor`), after feedback-orchestrator. slices 1-4 unchanged.

### ily/23 preserved

all 5 decisions from ily/23 are intact. deterministic backbone sacred. narrow evaluator for 5 decision points. no in-loop subagent. conservative fallback. Hermes dogfood is shadow only.
